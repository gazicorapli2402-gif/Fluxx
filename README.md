# 🚀 Flux - Modern Kanban & Task Manager

Flux, verimliliğinizi artırmak için tasarlanmış modern, hızlı ve şık bir görev yönetim uygulamasıdır. Kanban board yapısı, zaman takibi ve sesli komut desteği ile işlerinizi organize etmeyi kolaylaştırır.

## ✨ Özellikler

- **📊 Kanban Board**: Görevlerinizi "Yapılacak", "Devam Eden" ve "Tamamlandı" olarak sürükleyip bırakın veya tek tıkla taşıyın.
- **⏱️ Gelişmiş Zamanlayıcı**: Her görev için harcanan süreyi gerçek zamanlı olarak takip edin.
- **🎤 Sesli Komut**: Sesinizi kullanarak hızlıca yeni görevler ekleyin.
- **🔔 Hatırlatıcılar & Bildirimler**: Görev süreleri yaklaştığında tarayıcı bildirimleri alın.
- **📈 Analiz Paneli**: Toplam görev sayısı ve zaman dağılımı gibi verileri anlık olarak görün.
- **🎨 Modern Tasarım**: Glassmorphism estetiği ve dinamik animasyonlarla premium bir kullanıcı deneyimi.
- **💾 Yerel Kayıt**: Tüm verileriniz `localStorage` üzerinde güvenle saklanır, sayfa yenilense de kaybolmaz.

## 🚀 Başlangıç

### Gereksinimler

- Node.js (v14 veya üzeri)
- npm veya yarn

### Kurulum

1. Depoyu klonlayın:
   ```bash
   git clone https://github.com/KULLANICI_ADINIZ/Flux.git
   ```

2. Proje dizinine gidin:
   ```bash
   cd Flux
   ```

3. Bağımlılıkları yükleyin:
   ```bash
   npm install
   ```

4. Uygulamayı başlatın:
   ```bash
   npm start
   ```

Uygulama varsayılan olarak `http://localhost:3000` adresinde çalışacaktır.

## 🛠️ Kullanılan Teknolojiler

- **React**: Modern kullanıcı arayüzü bileşenleri.
- **Vanilla CSS**: Özel tasarım ve performanslı animasyonlar.
- **Web Speech API**: Ses tanıma özellikleri.
- **Notification API**: Tarayıcı tabanlı hatırlatıcılar.
